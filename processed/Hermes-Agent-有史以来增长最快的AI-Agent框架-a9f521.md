# Hermes Agent - 有史以来增长最快的AI Agent框架

> Hermes Agent是由Nous Research开发的开源AI Agent框架，定位于自主智能体而非聊天框。核心差异化在于三层记忆系统（会话/持久/Skill）+闭合学习循环，实现经验沉淀和能力进化。相比竞品Claude Code（局限于代码仓库）和OpenClaw（静态记忆），Hermes的Skill会随使用自动改进。内置微信原生支持、定时任务、子智能体等企业级功能。当前在OpenRouter排名全球第二，CLI Agent排名第一，GitHub星标近6万。

## 核心观点

- Hermes Agent是Nous Research开发的开源框架，GitHub星标近6万，两个月内达成
- 采用三层记忆系统（会话记忆、持久记忆、Skill记忆）实现跨会话学习
- 内置闭合学习循环：策划记忆→自主创建Skill→Skill自改进
- 支持原生微信连接（扫码连接私聊群聊），不需要公网IP
- 通过子智能体和定时任务实现自动化，支持自然语言设置定时任务

## 关键洞见

- 与Claude Code和OpenClaw不同，Hermes的核心优势是记忆持久化和Skill自动进化，使用越久越精准
- 传统Agent工具用完归零，Hermes打破了这一局限，让Agent从一次性工具变为长期搭档
- Skill是markdown文件，可读可编辑可复用，形成可积累的方法论资产

## Hermes 下一步行动

- 在本地环境安装Hermes Agent进行实际功能测试，重点验证三层记忆系统的实际效果
- 对比Hermes与现有Claude Code等工具在代码开发场景的差异，评估工作流优化空间
- 检查微信集成功能的稳定性和延迟表现，评估实际办公场景可用性
- 研究Skill文件格式和社区Skills Hub，探索自定义Skill开发可能性
