# Hermes agent 高级玩法

> AI代理Hermes的多场景高级配置与工作流优化指南

## 核心观点

- 通过Profiles实现配置、记忆、技能、会话完全隔离
- 固定工作目录可避免任务错乱，多项目用profile隔离
- 子Agent协作可用主模型统筹+子模型分工降低成本
- Checkpoints+Rollback机制提升AI改代码安全性
- Cron自动化可定时执行新闻抓取、日报生成等任务
