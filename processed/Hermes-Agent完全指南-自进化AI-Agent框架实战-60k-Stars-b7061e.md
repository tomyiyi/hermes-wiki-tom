# Hermes Agent完全指南：自进化AI Agent框架实战（60k Stars）

> 自进化AI Agent框架，内置学习循环支持技能自创与跨session持续进化

## 核心观点

- Hermes是唯一内置学习循环的AI Agent，能自创技能并持续进化
- 三层记忆系统：会话、Skills、用户模型，支持跨session持久化
- 支持多平台接入（Telegram/Discord等）和任意模型切换
- Skills系统可复用可分享，已形成agentskills.io生态
- 基于Agent循环架构：规划→执行→反思→记忆更新
