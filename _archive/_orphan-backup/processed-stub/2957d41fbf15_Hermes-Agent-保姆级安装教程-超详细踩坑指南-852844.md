# 2957d41fbf15_Hermes Agent 保姆级安装教程（超详细踩坑指南）

> Hermes Agent内置循环学习系统，可自动进化技能

## 核心观点

- Hermes Agent由Nous Research开发，GitHub星标超6万，增长迅猛
- 核心机制为内置循环学习系统，自动复盘并固化技能为Skill
- 对比OpenClaw：Hermes自动化程度高，OpenClaw可自定义性更强
- 安装需注意网络问题，Windows需WSL2环境，依赖安装有坑
- 小米MiMo免费套餐实为陷阱，需订阅Nous Portal才能使用
