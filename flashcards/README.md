# Hermes Wiki 闪卡库

本目录包含 hermes-wiki 核心知识的闪卡，用于 Obsidian Spaced Repetition 插件复习。

## 导入方法

1. 打开 Obsidian，进入 hermes-wiki 保险箱
2. 打开 "Spaced Repetition" 插件
3. 点击设置，配置闪卡文件夹为 `flashcards/`
4. 插件会自动扫描并导入所有闪卡

## 闪卡分组

| 文件 | 内容 | 优先级 |
|------|------|--------|
| Hermes核心概念.md | WAL协议、工作流、Vault结构 | P0 |
| OPC社区项目.md | V11方案、双轨人群、八大服务 | P0 |
| AI_LLM理论.md | Ilya/Yudkowsky/mmguo方法论 | P1 |
| gstackSkills.md | 高价值Skills、工作流、组合分工 | P1 |

## 复习频率建议

- Hermes核心概念：每天
- OPC社区项目：每天
- AI_LLM理论：每2天
- gstackSkills：每3天
