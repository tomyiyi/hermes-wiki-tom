# Deep Compression 三层检验

- Q1 沉默的维度：作者为什么说这个，而不说别的？
- Q2 可证伪性：什么事实或论证可以推翻这个结论？
- Q3 隐藏推论：能推导出一个作者没有明确说、但从逻辑上必然成立的结论吗？

# Trident 三视角分析是什么？

- 🔭 Framework Architect（框架构建者）：扩展、延伸、找因果结构
- ⚔️ Critical Thinker（批判者）：找弱点、找漏洞、找未声明的假设
- 🔧 Practitioner（实践者）：落地行动、具体步骤、可操作性
