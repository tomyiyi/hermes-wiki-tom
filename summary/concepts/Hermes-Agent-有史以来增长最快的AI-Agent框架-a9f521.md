# Hermes Agent - 有史以来增长最快的AI Agent框架

> **Hermes Agent是一款开源AI Agent框架，通过三层记忆系统和Skill机制实现持续学习和经验沉淀。**

Hermes Agent代表了AI Agent从「用完归零」到「持续进化」的范式转变，其三层记忆系统和Skill机制让工具本身具备学习能力，差异化优势明显。但实际生产环境效果和长期维护成本仍待验证。
