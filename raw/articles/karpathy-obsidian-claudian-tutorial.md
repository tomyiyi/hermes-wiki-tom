# Karpathy AI + Obsidian 知识库教程

## 工具链
- Obsidian（笔记软件）
- Claude Code（AI 编码助手）
- Claudian（Obsidian 插件，将 Claude Code 接入 Obsidian 图形界面）

## 安装步骤

### 1. 安装 Obsidian
下载地址：obsidian.md

### 2. 安装 Claude Code
Claude Code 链接（略），网上安装教程很多

### 3. 安装 Claudian 插件
- BRAT 插件 → 添加插件 → 输入：
```
https://github.com/YishenTu/claudian
```

### 4. 安装 Obsidian 必备 5 个 Skill
由 Obsidian CEO 亲自制作
安装方式：直接把链接扔给 Claude Code
```
https://github.com/kepano/obsidian-skills
```

## 使用流程

### 1. 创建知识库文件夹结构
1. 打开 Obsidian 新建一个仓库
2. 把 karpathy 的方法论发给 AI
3. AI 自动创建所有文件夹和对应文件

### 2. karpathy 方法论文件（复制发给 AI）
```
# LLM Wiki - A pattern for building personal knowledge bases using LLMs

## The core idea
Most people's experience with LLMs and documents looks like RAG...
```

## 关键要点
- AI 负责建立文件夹结构
- AI 负责内容分发
- 人类负责确认方案
- 图形化界面比命令行更直观
