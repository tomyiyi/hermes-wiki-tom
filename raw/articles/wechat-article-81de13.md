Hermes Agent 保姆级安装教程（超详细踩坑指南） 原创 陈虾仁 陈虾仁 婚礼策划 在小说阅读器读本章 去阅读 在小说阅读器中沉浸阅读 这几天的 AI 圈，被 Hermes Agent 给刷屏了! 到处都有人拿它跟 OpenClaw 放一起对比。有的朋友更是动作麻利，直接把 OpenClaw 卸了，毫不犹豫地投入了 Hermes 的怀抱。 这玩意到底是真神，还是只是一阵风？那在正式进入保姆级教程前，我先带你带你扒一扒它的底裤，最后才把它装到你的电脑上。 “爱马仕”也出 Agent 了？ 别误会，此“爱马仕”非彼爱马仕，不是卖包的那个，而是 Nous Research 搞出来的当红炸子鸡 —— Hermes Agent。 这项目刚上 GitHub 没多久，星标已经超过 6 万了，趋势榜榜首也是随便拿捏。 它凭什么能硬刚 OpenClaw？ 我们平时用的 ChatGPT、豆包、DeepSeek，包括大部分 Agent，都有个共同点： 你给指令，它干活；干完拉倒，下次重来。 但 Hermes 的脑回路不一样，它有一个王炸级别的核心机制： 内置循环学习系统（Built-in learning loop）。 它帮你干完一件事后，不会拍拍屁股走人，而是会回头“复盘”：刚才这活儿干得咋样？哪里能优化？然后，它会把这套流程固化成一个 Skill（技能） 。下次再遇到类似任务，它不从零开始，直接调取 Skill 干活。你要是中间纠正了它，它还会自动迭代版本。 你用得越久，它进化的越牛。 这时候你再看 OpenClaw，差别就出来了。OpenClaw 更像是一套强大的 DIY 乐高，你要自己接工具、写配置，一点点把它“养”大；而 Hermes 呢，是你用着用着，它自己就把本事长齐了。不过听句劝，真没必要二选一。OpenClaw 负责搭框架、接平台，Hermes 负责在底层把活儿越做越顺， 强强联手才是成年人的选择。 保姆级安装教程 接下来，我们直接把它装起来！官方号称 60 秒就能搞定，但实测下来，几乎不可能。 具体安装步骤 1、跑命令行 不管你是 Linux 还是 Mac，直接复制官方以下这行命令。如果你是 Windows用户， 切记要在WSL2环境下安装， 别在原生命令行里硬怼。 复制： curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash 打开终端，粘贴，回车。 2、解决网络问题 接下来它会自动检查并安装各种依赖环境（UV package 等）。 注意，重点来了： 如果你发现画面卡住不动了，别怀疑人生， 99% 是网络问题 。 遇事不决，先停掉，调整一下你的魔法网络（比如强制走终端代理），然后再试一遍。折腾这种新工具，解决网络问题本身就是乐趣的一部分，多点耐心。 3、环境配置 网络通了之后就快了。你会看到它噼里啪啦一顿跑，装依赖、配虚拟环境、装 NodeJS 相关包，甚至还会帮你装自动化浏览器引擎。 避坑指南： 如果中间提示某些安装失败，比如浏览器引擎安装失败，别慌，一般不影响核心功能，让它继续往下走就行，后面可以再让它重新安装。 4、导入与初始化 到这一步，其实已经装成功了，接下来是引导配置。 要不要导入 OpenClaw 配置？ 如果你电脑里有 OpenClaw，它会问你要不要把 Soul、Memory 这些文件导进来。如果你嫌配置麻烦，可以直接导入，但官方也提示了，不会 1:1 还原。我的建议是： 选 No 。咱们先搞个干净的新环境摸摸它的脾气，配置以后再说。 安装模式： 选默认的快速安装（Quick Install）就行，直接回车。 5、搞定模型 API (别被免费套路了) 它会让你选 AI 供应商。 可能你最近看到新闻说“小米MiMo V2 Pro接入 Hermes 免费用两周”。 这里有个大坑！ 你选了 Nous Portal 去接小米模型，最后会发现：必须得先订阅 Nous Portal，才能享受所谓的“免费两周”。虽然它有 0 元的订阅，但要填国际卡的信息，而且还可能被拒，所以对于国内大部分人，基本等于用不上这个免费额度。 最简单的解法：用你自己的APIKey。 下拉选单，选择 More。 找一个你自己有的模型，比如 MiniMax（记得选 China 版本，别选成国际版，我第一次就是选了MiniMax global direct API这个版本所以不成功）。 去你MiniMax开放者平台的后台，把你自己的 API Key 粘进去，回车保存。 选一个你喜欢的模型（比如 2.7），搞定！ 6、设置你你跟Hermes的对话工具 你可以先“Skip”跳过，也可以现在就设置，选择 Set up messaging now 你可以选择你想要的通信工具，比如说Telegram、Discord、WhatsApp、微信等 我就先跳过不设置了，你们需要设置的，按指引步骤一步步绑定就行 7、见证奇迹的时刻 模型设置成功后，重新用Hermes命令进入主界面 成功登录 我们先来句灵魂拷问：“你是谁？” 它居然回答“我是Claude Code” 别管它叫啥了，直接派活：“ 搜索今天 5 条热门 AI 新闻给我。 ” 只见它一顿操作，自动调用 Google News 搜索，不一会儿就把今天最新的 5 条资讯给你扒出来了。 搞定！你已经成功征服了这台“爱马仕”。 灵魂拷问：你到底要不要装？ 折腾了一圈，最后我给你交个底： 不是所有人都需要现在就上车。 就拿我自己来说，我那台 Mac Mini M4 上跑着一套用 OpenClaw 搞的自动化新闻抓取和播客系统，每天稳如老狗。我干嘛非得为了赶时髦把它拆了换 Hermes？完全没必要。 工具的本质是 降本增效 。如果你手头的流程已经跑得很顺了，换新工具并不会让你立刻飞升。 现在很多人一看到新 AI 出了就焦虑，生怕落后，但我劝你： 打死不要做“工具收集癖”， 今天学这个，明天追那个，最后啥也没沉淀下来。先把你现在手头的 AI 玩到极致，遇到真正的瓶颈了，再去换新工具。 用好一个，你已经超越 99% 的人了。 留个悬念 从安装到解决网络、API 配置，今天先把 Hermes Agent 的底座给你搭好了，希望能帮你少走几步弯路。 至于它到底适不适合长期挂在后台？Skills 自动进化的能力在复杂工作流里表现到底怎么样？后续我会做更深入的压力测试，到时候再给大家出一期详细汇报。 赶紧把你的Hermes装起来吧！ 遇到什么奇葩 Bug，或者觉得特别好用的地方，直接把感受砸在评论区，我们一起探讨。 别忘了关注我，后面我再接着拆解更多好玩的 AI
