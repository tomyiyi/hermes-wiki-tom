# 股票分析系统架构文档

> 创建时间：2026-04-12
> 版本：v6.0

---

## 一、系统架构总览

```
用户输入：「分析股票 300436」
        ↓
    OpenClaw
        ↓
┌─────────────────────────────────────────┐
│           Skill Layer (调度层)            │
│  stock-analysis (v6分析)                │
│  a-stock-market (实时行情)               │
│  eastmoney-stock (东方财富)              │
│  stock-recommend (选股推荐)              │
└─────────────────────────────────────────┘
        ↓
┌─────────────────────────────────────────┐
│           Script Layer (执行层)           │
│  stock-report-v6.py (数据采集核心)        │
│  stock-daily-report.py (每日推送)         │
│  stock-backtest.py (回测引擎)            │
│  stock-sina.py (新浪实时行情)            │
└─────────────────────────────────────────┘
        ↓
┌─────────────────────────────────────────┐
│           Data Layer (数据层)             │
│  新浪财经 API (K线/实时)                │
│  东方财富 API (资金流向)                 │
│  akshare (财务数据)                      │
│  prophet (预测模型)                      │
└─────────────────────────────────────────┘
        ↓
┌─────────────────────────────────────────┐
│           Output Layer (输出层)           │
│  飞书文档 (docx)                        │
│  Markdown (本地)                          │
│  JSON (数据)                             │
└─────────────────────────────────────────┘
```

## 二、Skills 清单


## 四、数据源

| 来源 | 数据类型 | 接口 |
|------|---------|------|
| **新浪财经** | K线/实时价格/技术指标 | `hq.sinajs.cn` |
| **东方财富** | 资金流向/超大单/散户 | `push2.eastmoney.com` |

| **prophet** | 时间序列预测/未来5日价格 | Facebook开源 |

## 五、v6 分析报告 10 章节

```
一、核心定位与基本盘
   - 股票代码/名称
   - 当前价格/涨跌幅
   - 综合评级

二、MyTT技术指标全景
   - 均线系统 (MA5/10/20/60)
   - RSI (12日)
   - MACD柱
   - CCI顺势指标
   - 乖离率 (BIAS5/20)
   - BOLL布林线

三、主力资金深度拆解
   - 主力净流入
   - 超大单净流入
   - 大单/中单/散户

四、财务三表诊断
   - EPS (每股收益)
   - ROE (净资产收益率)
   - 资产负债率
   - 毛利率
   - 营收增长率
   - 净利润增长率

五、Prophet趋势预测
   - 未来5日预测价格
   - 趋势方向判断

六、回测验证
   - MA金叉策略收益率
   - RSI超买超卖策略
   - MACD交叉策略
   - BOLL突破策略
   - 最优策略推荐

七、次日情景演练
   - 上涨概率 (🐂)
   - 震荡概率 (📊)
   - 下跌概率 (🐻)

八、操作建议
   - 持仓者建议
   - 观望者建议
   - 三周期策略 (超短/波段/中线)

九、风险提示
   - 财务风险
   - 市场风险
   - 仓位建议

十、综合评级
   - 五维评分 (技术/资金/基本面/趋势/综合)
   - 一句话总结
```

## 六、工作流示例

### 完整分析流程

```
用户：「分析股票 300436」

1. OpenClaw 接收请求
   ↓
2. 调用 stock-analysis Skill
   ↓
3. 运行 stock-report-v6.py
   - 获取K线数据
   - 计算技术指标
   - 获取资金流向
   - 获取财务数据
   - Prophet预测
   - 回测验证
   ↓
4. 解析数据生成 v6 格式
   ↓
5. 写入飞书文档
   ↓
6. 返回链接给用户
```
