9k。Hermes Agent 的杀手锏不是跑得更快，而是会自我进化——你用得越久，它就越懂你。核心通过三大系统实现：Memory（记住你）、Skill（自动从任务里总结出可复用技能）、Nudge Engine（主动提醒、主动学习）。它不是那种每次都把你当陌生人的对话机器人，而是一个真正会成长的数字助手。更关键的是，它原生支持微信接入——装好之后直接在微信里发消息，就能远程指挥它写周报、搜资料、执行任务。但是！跟着官方安装指令跑一遍，在国内大概率卡死在第一步——git clone 超时、pip install 挂掉、npm 依赖下不动、Playwright 浏览器下载失败……每一个环节都能让你怀疑人生。我已经帮大家把这六个坑全踩完了，今天一次性给出完整的一键脚本 + 微信接入全流程，跟着做，你也能拥有一个随叫随到、还会主动成长的 AI 助手。一、在这之前：先确认环境开始之前，确保你的机器上装了这三样东西：依赖版本要求检查命令Python3.11+python3 --versionGit不限git --versionNode.js22+node Bot，并拿到以下信息：WEIXIN_ACCOUNT_ID：机器人账号ID，格式如 xxx@im.botWEIXIN_TOKEN：对应的 token，通常和 account_id 组合在一起，格式 account_id:secretWEIXIN_BASE_URL：接口地址，默认为 https://ilinkai.weixin.qq.comWEIXIN_CDN_BASE_URL：CDN 地址，保持默认 https://novac2c.cdn.weixin.qq.com/c2c（具体申请流程因平台而异，搜索“微信客服机器人 token 获取”即可找到最新指引，通常几分钟就能搞定。）5.2 将凭证写入 .env 文件打开 ~/.hermes/.env，添加以下内容（请替换为你的真实凭证）：# ---------- 微信通道 ----------WEIXIN_ACCOUNT_ID=055ec21e2a4b@im.botWEIXIN_TOKEN=055ec21e2a4b@im.bot:你的secretWEIXIN_BASE_URL=https://ilinkai.weixin.qq.comWEIXIN_CDN_BASE_URL=https://novac2c.cdn.weixin.qq.com/c2c# 访问控制策略WEIXIN_ALLOW_ALL_USERS=true# 是否允许所有微信用户发消息（建议先 true 测试）WEIXIN_ALLOWED_USERS= # 允许的用户 openid，多个用逗号分隔WEIXIN_GROUP_ALLOWED_USERS= # 群聊白名单WEIXIN_HOME_CHANNEL= # 主频道，留空即可# 安全策略WEIXIN_DM_POLICY=pairing # 私聊策略：pairing=新用户需审批，open=开放，disabled=关闭WEIXIN_GROUP_POLICY=disabled # 群聊策略：一般建议 disabled注意：WEIXIN_DM_POLICY=pairing 是最推荐的模式。当有新用户第一次给你发消息时，Hermes 会返回一个 8 位配对码，只有你在服务器上手动 approve 之后对方才能对话，有效防止陌生人滥用。5.3 启动微信网关配置保存后，在服务器上运行：hermes gateway你会看到类似 [weixin] Connected 的日志输出，说明微信通道已成功上线。保持这个终端窗口一直运行，建议配合 screen 或 tmux 使其在后台常驻：screen -S hermeshermes gateway# 按 Ctrl+A 然后 D 分离5.4 处理新用户配对审批当有陌生人（或你自己第一次）在微信给机器人发消息时，会收到这样一条回复：Hi~ I don't recognize you yet! Here's your pairing code: YK5Y566L Ask the bot owner to run: hermes pairing approve weixin YK5Y467L这时你需要回到服务器，在hermes执行提示的那条命令：hermes pairing approve weixin YK5Y467L执行完即刻生效，对方再发消息就能正常对话了。如果你希望自己以后不用再审批，可以把你的 openid 填入 WEIXIN_ALLOWED_USERS 里，或者在首次审批后改为 open 模式。5.5 安全命令审批（遇到时别慌）在日常聊天中，如果 Hermes 要执行一些它认为存在潜在风险的命令（例如访问没有带 https:// 前缀的网址），它会在微信里暂停并发送一个审批请求：⚠️ Dangerous command requires approval: curl -s "wttr.in/..." Reply /approve to execute, /approve session to approve this pattern for the session, /approve always to approve permanently, or /deny to cancel.你只需在微信对话框里直接回复对应的指令即可：回复作用/approve仅本次放行/approve session本次会话内同类命令自动放行/approve always永远不再拦截该模式的命令/deny拒绝执行这是 Hermes 的安全红线，防止它被恶意链接诱导执行危险操作，建议保持开启。六、验收时刻所有配置完成后，运行：hermes doctor如果你看到下面这些绿勾，恭喜你，一个完全体 Hermes Agent 已经上线：绑定了微信通道，我们就可以直接在微信给hermes发送指令，进行沟通了，让它介绍一下自己。现在，你可以在微信里随时对它说：“帮我查一下今天北京的天气”“每天下午6点提醒我开会”它都会像一个真正的助理一样去执行。最后让它写一个小程序,直接写一个C语言课设图书管理系统，从编写代码，编译都执行了，一次性完成，效果显著。七.写在最后Hermes Agent 之所以能刷屏，不仅仅是因为跑得快或者功能全——更重要的是它带来了一个真正会 “自我进化” 的 AI 形态。长久记忆、技能自主生成、持续优化，越用越智能，越用越贴合你的需求。而这篇文章的使命，就是帮你在国内的网络环境下，跳过所有不必要的坑，一步到位跑起来，并且直接住进微信里。如果在安装过程中遇到任何问题，欢迎在评论区留言，会尽量帮大家解答。本期主编：飞鸟 & 公子龙
